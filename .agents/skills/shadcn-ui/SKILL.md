# Shadcn UI Integration Skill

This skill allows Antigravity to automatically install, configure, and maintain Shadcn UI components inside this repository using CLI utilities and modern standards.

## Usage Guidelines
- Always use `@/components/ui` path alias for imports.
- Make sure components are compatible with the Tailwind version set in the project.
- Check and install required dependencies like `lucide-react`, `clsx`, or `tailwind-merge` when adding components.
